Data Scientist Nanodegree¶

Project: Write a Data Science Blog¶

Business Understanding¶

In this project, the objective is to pick a dataset of your own choice and answer at least 3 questions related to business or real-world applications of how the data could be used.

Questions To Be Answered

Q1. What the dominated gender in data science and what is the highest age ratio?

Q2. What the most compensation the data scientists can get?

Q3: What media sources data scienticts get the information from?

Data Understanding¶

The dataset that I have prepared for this project, it is Kaggle’s third annual survey. it includes 19,717 Kaggle members' respondents in the survey. To answer the 3 questions in interest, I'll focuse on the members that are currently employed as data scientists.

# function to count the values in Dataframe
def value_counts(df, column, normalize= True, rename= 'Percentage', return_percent= True):
    if rename is not None and return_percent:
        mod_df= df[column].value_counts(normalize=normalize).rename(rename).mul(100).reset_index()
        mod_df= mod_df.rename(columns ={'index':column})
        return mod_df

def multiple_answers(limit, df, col, mod_name):
    dict_local = {}

    for i in range(1, limit+1):
        col_name = col + str(i)
        new_col_name = df[col_name].value_counts().reset_index().iloc[0,0]
        dict_local[new_col_name]  = df[col_name].value_counts().reset_index().iloc[0,1]

    new_df = pd.DataFrame(dict_local, index=[0])
    new_df = new_df.melt()
    new_df = new_df.sort_values(by='value', ascending=False)
    new_df = new_df.rename(columns= {'variable':mod_name, 'value':'Count'})
    return new_df

def combine_row_Data_Scientist(df_Data_Scientist, df_row):
    df_Data_Scientist['Job Title'] = Job_Title
    df_row['Job Title'] = 'Rest of other job titles'
    concat = pd.concat([df_Data_Scientist,  df_row], axis = 0).reset_index(drop=True)
    column = list(df_Data_Scientist.columns)[0]
    length_1 = list(df_Data_Scientist[column].unique())
    length_2 = list(df_row[column].unique())
    dict_more=[]
    if(len(length_1) == len(length_2)):
        return concat
    elif(len(length_1) > len(length_2)):
        not_present = []
        for col in length_1:
            if col not in length_2:
                not_present.append(col)
        for col in not_present:
            dict_more.append([col, 0, 'Rest of other job titles'])
            
    elif(len(length_1) < len(length_2)):
        not_present = []
        for col in length_2:
            if col not in length_1:
                not_present.append(col)
        for col in not_present:
            dict_more.append([col, 0, Job_Title])
